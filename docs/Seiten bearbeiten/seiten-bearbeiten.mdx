# Seiten bearbeiten

Dies ist vermutliche die wichtigste Funktion für alle. Alle Referate, Servicebetriebe und das StuPa besitzen eigene Unterseiten auf der Website, die von euch selbst verwaltet werden können. Um eure Seite zu bearbeiten, gibt es zwei Möglichkeiten:

**A)** Geht auf der AStA-Website auf eure Seite, die ihr bearbeiten wollt, und klickt in der Leiste oben auf „Seite bearbeiten“.

![](/img/guide-pics/image8.png)

**B)** Klickt im Adminbereich in der Leiste links auf „Seiten“.

![](/img/guide-pics/image9.png)

Es wird euch eine Übersicht der Seiten angezeigt, die ihr verfasst habt/die ihr verwalten könnt. Im Falle von Referaten sollte das nur jeweils eine sein. Klickt bei dieser dann auf „bearbeiten“.

![](/img/guide-pics/image10.png)

Egal, welchen Weg ihr gewählt habt, solltet ihr nun im Editor landen.

## Der Seiten Editor

![](/img/guide-pics/image11.png)

Die Benutzung des Editors ist relativ simpel. Im Großen Feld in der Mitte schreibt ihr den Text für eure Seite. Dabei stehen euch verschiedene sogenannte Blöcke zur Verfügung. Schreibt ihr einen Absatz, ist das zum Beispiel ein Block. Diesen könnt ihr mit den klassischen Formatierungsoptionen (Fett/Kursiv/Unterstrichen) versehen, ihr könnt Verlinkungen einfügen, etc...

Ein Block kann aber zum Beispiel auch ein Bild sein, welches ihr in euren Beitrag einfügt, oder ein in die Seite eingebettetes YouTube-Video. Der Editor bietet euch hier eine Fülle von Optionen, die an dieser Stelle nicht alle erklären werden können. Das meiste sollte allerdings selbsterklärend sein, spielt also gerne einfach ein wenig rum. Habt ihr eine Frage zu einem bestimmten Block, könnt ihr gerne den Webmaster anschreiben.

### Bilder einbinden
Da die wohl wichtigste Funktion ist, Bilder in eure Seiten einzufügen, wird diese hier noch einmal genauer erklärt.

import image13 from '@site/static/img/guide-pics/image13.png';

<ImageAside src={image13} third>
	Wählt ihr „Bild“ als Typ aus, erscheint der Block mit einer kurzen Beschreibung, wie ihr ein Bild einfügt. Im Grunde macht ihr dann genau das: Klickt entweder auf „Hochladen“ und wählt ein Bild von eurem Computer aus, welches dann auf den Server hochgeladen wird, oder klickt auf
</ImageAside>


„Mediathek“ und wählt ein Bild aus, welches bereits einmal hochgeladen wurde. In beiden Fällen sollte das Bild nun im Editor angezeigt werden. Direkt darüber habt ihr die Möglichkeit, verschiedene Filter oder einen Textumfluss einzustellen, das heißt, dass falls das Bild klein genug ist, Text links oder rechts neben diesem angezeigt werden kann.

:::info
Die Größe des Bildes kann angepasst werden, indem ihr im Editor an den blauen Punkten rechts und unter dem Bild zieht.
:::

Unter dem Bild habt ihr außerdem die Möglichkeit, eine Beschriftung einzugeben, die dann wie im Editor auch auf der Seite angezeigt wird. Praktisch, falls ihr in ein paar Wörtern nochmal auf das Bild eingehen wollt, ohne es in euren eigentlichen Text zu schreiben.

## Seite veröffentlichen
Um die Änderungen an eurer Seite zu bestätigen und sie zu veröffentlichen, klickt oben rechts auf den blauen Button „Aktualisieren“.

:::caution Achtung
Bevor ihr die Seite veröffentlicht/aktualisiert, klickt einmal auf „Vorschau“ und überprüft, dass eure Änderungen zu keinem unerwünschten Effekt geführt haben und alles auch wirklich so aussieht, wie ihr es möchtet. Denkt dran, dass sobald ihr die Seite aktualisiert, jede\*r, die\*der AStA-Website besucht, diese Änderungen sehen kann.
:::

:::warning Achtung
Klickt auf keinen Fall auf „Auf Entwurf umstellen“. Damit wird die Veröffentlichung eurer Seite Rückgängig gemacht, was bedeutet, dass jede\*r Besucher\*in der Seite eine Fehlermeldung erhält.
:::
