## Der Seiteneditor

![](/img/guide-pics/image11.png)

Im Seiteneditor erstellt ihr eure Seite mit Hilfe der Wordpress Blocks so, wie ihr sie haben wollt. Im großen Feld in der Mitte schreibt ihr den Text für eure Seite. Schreibt ihr einen Absatz, ist das zum Beispiel ein Block. Eine Überschrift wäre auch ein Block. ABsätze könnt ihr mit den klassischen Formatierungsoptionen (Fett/Kursiv/Unterstrichen) versehen, ihr könnt Verlinkungen einfügen, etc...

Ein Block kann aber zum Beispiel auch ein Bild sein, welches ihr in euren Beitrag einfügt, oder ein in die Seite eingebettetes YouTube-Video, ein PDF oder ein Formular, das Daten abfragt und sammelt.

:::info
Das Meiste der Funktionen sollte selbsterklärend sein, spielt also gerne einfach ein wenig rum. Bei Fragen, wie spezifische Vorstellungen von euch umgesetzt werden können, oder wenn ihr Hilfe braucht, meldet euch jederzeit gerne an den/die Webmaster*in (webmaster@asta-lueneburg.de).
:::

### Bilder einbinden
Da die wohl wichtigste Funktion ist, Bilder in eure Seiten einzufügen, wird diese hier noch einmal genauer erklärt.

Über einen Klick auf das Plus zwischen zwei Blöcken fügt ihr einen neuen Block „Bild“ hinzu. Ihr solltet jetzt folgendes Bild angezeigt bekommen:

![](/img/guide-pics/image13_Bildeinfuegen.png)

Klickt entweder auf „Hochladen“ und wählt ein Bild von eurem Computer aus, welches dann auf den Server hochgeladen wird, oder klickt auf „Mediathek“ und wählt ein Bild aus, welches bereits einmal hochgeladen wurde.

:::caution Wichtig
Wenn ihr das Bild ausgewählt habt, gebt bei Alternativtext (Alt-Text, mehr dazu auch weiter unten) gerne eine Kurzbeschreibung des Bildes an, sollte es relevant für euren Inhalt sein. Dann können Screenreader für Menschen mit Sehbeeinträchtigungen diesen Alternativtext vorlesen.
:::

**Das Bild sollte jetzt im Editor angezeigt werden.**

Direkt darüber habt ihr die Möglichkeit, verschiedene Filter oder einen Textumfluss einzustellen, das heißt, dass falls das Bild klein genug ist, Text links oder rechts neben diesem angezeigt werden kann.

### Mediendateien einfügen
Neben Bildern können auch andere Mediendateien, wie z. B. PDFs und Videos in der Website eingefügt werden. Die Mediendatei ladet ihr dafür am Besten in eurem Dashboard unter „Medien" hoch.

Disclaimer: auch Bilder können über die Mediathek hochgeladen werden und dann im Seiteneditor eingefügt werden.

## Seite veröffentlichen
Um die Änderungen an eurer Seite zu bestätigen und sie zu veröffentlichen, klickt oben rechts auf den blauen Button „Aktualisieren“.

:::caution Achtung
Bevor ihr die Seite veröffentlicht/aktualisiert, klickt einmal auf „Vorschau“ und überprüft, dass eure Änderungen zu keinem unerwünschten Effekt geführt haben und alles auch wirklich so aussieht, wie ihr es möchtet. Denkt dran, dass sobald ihr die Seite aktualisiert, jede\*r, die\*der AStA-Website besucht, diese Änderungen sehen kann.
:::

:::warning Achtung
Klickt auf keinen Fall auf „Auf Entwurf umstellen“. Damit wird die Veröffentlichung eurer Seite Rückgängig gemacht, was bedeutet, dass jede\*r Besucher\*in der Seite eine Fehlermeldung erhält.
:::
