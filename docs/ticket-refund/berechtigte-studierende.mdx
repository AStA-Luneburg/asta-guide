# Berechtigte Studierende

:::info
Damit Studierende ihre Rückerstattungen beantragen können, verwaltet das System eine Liste an berechtigten Studierenden.
:::

## Hinzufügen, Bearbeiten und Löschen von Studierenden

- Im Admin-Dashboard können Studierende hinzugefügt werden. 
- Hierfür muss eine Matrikelnummer, ein Name und eine E-Mail angegeben werden.
  - Die E-Mail muss mit dem festgelegten E-Mail-Suffix enden.
- Die Daten können im Admin-Dashboard auch bearbeitet und gelöscht werden.
- Wenn der/die Studierende einen Antrag gestellt hat, ist dieser hier auch einsehbar.

## Massenimport von Studierenden

Ein Massenimport wird nicht direkt unterstützt und muss über externe Tools wie phpMyAdmin getan werden.
