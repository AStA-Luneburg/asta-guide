## Anmelden in das Admin-Dashboard
Um sich in das Admin-Dashboard anzumelden, muss das normale Verifizierungs-Formular verwendet werden.

**Die Anmeldung funktioniert nur über E-Mail Adressen, die vorher vom Webmaster für das Dashboard freigeschaltet wurden.**

:::caution Achtung
Wenn du bereits mit deiner persönlichen Studi-Email angemeldet bist, musst du vorher auf "Abmelden" drücken, um dich mit dem Admin-Account anmelden zu können.
:::
