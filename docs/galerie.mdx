# Galerie

Alle Referate (und manche Servicebetriebe) haben Zugriff auf eine eigene Galerie, die in Seiten und Beiträge eingefügt werden kann.

Um Bilder zu dieser Galerie hinzuzufügen, klickt in der Seitenleiste des Administrationsbereiches auf *Galerie/Bilder hinzufügen* unter *Galerie*. Es öffnet sich die Seite zum Hochladen von Bildern.

![](/img/guide-pics/image31.png)

**Wählt unter *Galerie* den Namen eurer Organisation aus. Normalerweise sollte im Auswahlmenü auch nur diese auftauchen.**

:::caution Achtung
Solltet ihr aus irgendeinem Grund keine Galerie auswählen können, kontaktiert bitte den Webmaster und erstellt keine neue Galerie.
:::

Klickt anschließend auf *Dateien hinzufügen* und wählt Dateien von eurem Computer aus, die ihr in eure Galerie hochladen wollt. Habt ihr alle hinzugefügt, klickt auf *Upload beginnen*. Die Dateien werden nun hochgeladen. Sobald der Upload beendet wurde, solltet ihr eine Meldung erhalten und hinter jedem Bild sollte ein grüner Haken erscheinen. Bitte verlasst die Seite erst dann, da es sonst sein kann, dass nicht alle Bilder erfolgreich hochgeladen wurden.

## Bilder verwalten

Damit ihr die Informationen zu den gerade hochgeladenen Bildern bearbeiten könnt oder unerwünschte Bilder löschen könnt, klickt auf *Galerien verwalten* unter *Galerie* in der Seitenleiste. Ihr erhaltet nun eine Übersicht aller Galerien, wobei alle ausgegraut sein sollten außer eure eigene. Klickt ihr hier auf den Titel eurer Galerie, gelangt ihr zu der Seite, wo ihr eure Bilder verwalten könnt.

![](/img/guide-pics/image32.png)

Hier habt ihr eine Vielzahl an Möglichkeiten, um eure Bilder zu bearbeiten und dessen Informationen zu ändern. So könnt ihr zum Beispiel den Titel und eine Beschreibung eingeben, die später in eurer Galerie angezeigt wird. Möchtet ihr ein Bild aus der Galerie entfernen, klickt unter dem Bild auf *Löschen*.

![](/img/guide-pics/image33.png)

Ihr müsst eure Änderungen speichern, indem ihr entweder oben oder ganz unten auf der Seite auf *Änderungen speichern* klickt.

![](/img/guide-pics/image34.png)

## Galerie einem Beitrag/einer Seite hinzufügen
Damit die Galerie auf der Website angezeigt wird, muss sie einer Seite oder einem Beitrag hinzugefügt werden. Bearbeitet dafür eine Seite oder einen Beitrag, sodass ihr im Editor landet.

Fügt anschließend den Block *NextGEN Gallery* hinzu und klickt auf den großen Button *Add NextGEN Gallery*.

Es öffnet sich ein Fenster, in dem ihr eure Galerie hinzufügen könnt.

![](/img/guide-pics/image35.png)

Klickt ihr auf „Select a Gallery“ öffnet sich eine Auswahl mit den verfügbaren Galerien. Wählt hier die, die ihr einfügen wollt (in den meisten Fällen vermutlich eure eigene). Darunter habt ihr die Möglichkeit, verschiedene Darstellungsweisen für die Galerie auszuwählen. Die Bilder dort erklären eigentlich ganz gut, wie die verschiedenen Optionen aussehen.

:::info
Die Darstellungsweise „Basic TagCloud“ funktioniert nur, wenn ihr für eure Bilder Stichwörter eingetragen habt (siehe „Bilder verwalten“).
:::

Für eine bessere Benutzerfreundlichkeit wechselt vor dem Einfügen noch einmal kurz auf den Reiter „Customize Display Settings“ und setzt die Option „Aktiviere AJAX-Naviagtion“ auf „Ja“. Damit können Besucher*innen zwischen den einzelnen Seiten eurer Galerie navigieren, ohne dass die Seite neu geladen wird. Ihr könnt dort außerdem auch einstellen, wie viele Bilder pro Seite angezeigt werden sollen.

Seid ihr fertig, klickt ihr unten auf „Insert Gallery“. Die Galerie ist damit im Beitrag eingefügt, sie wird im Editor allerdings nicht wirklich angezeigt. Anschließend müsst ihr eure Seite/euren Beitrag aktualisieren. Damit wird die Galerie dort angezeigt.
