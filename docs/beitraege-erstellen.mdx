# Beiträge erstellen 

Ihr habt Neuigkeiten aus euren Referaten, Servicebetrieben oder dem StuPa, die ihr veröffentlichen wollt? Perfekt! Alle haben die Möglichkeit (und ihnen wird ans Herz gelegt!), sogenannte Beiträge zu veröffentlichen, die dann auf der Startseite angezeigt werden und so alle über Neuigkeiten aus dem AStA/StuPa informiert werden.

![](/img/guide-pics/image29.png)

Um einen Beitrag zu erstellen, klickt ganz einfach in der Leiste oben auf der AStA Website auf „Beitrag“ unter „Neu“. Es öffnet sich anschließend derselbe Editor, den ihr auch schon vom Bearbeiten/Erstellen eurer Seiten kennt. Und genau so funktioniert auch das Erstellen von Beiträgen. Auch hier stehen euch alle Blöcke zur Verfügung, um den Beitrag zu schreiben und aufzuhübschen. Vor dem Veröffentlichen müssen allerdings noch ein paar Einstellungen getätigt werden.

## Kategorie

Euer Beitrag muss einer Kategorie zugeordnet werden, in den allermeisten Fällen der Kategorie eurer Organisation. Öffnet dafür die Box „Kategorien“, indem ihr auf deren Titel klickt. Es erscheint eine Übersicht aller Kategorien. Wählt dort den Namen eurer Organisation aus, indem ihr auf die Box neben deren Namen klickt.

:::info
Aktuell habe nicht alle Servicebetriebe eine Kategorie. Möchtet ihr einen Beitrag
posten, findet aber eure Kategorie nicht, kontaktiert bitte erst den Webmaster und wählt keine andere Kategorie aus. Ich lege euch dann eine neue Kategorie an.
:::

:::caution Wichtig
In fast allen Fällen sollte der Haken für die Kategorie „Allgemein“ herausgenommen werden, um so eine bessere Übersichtlichkeit bei der Sortierung von Beiträgen zu erreichen.
:::

## Beitragsbild

Jeder Beitrag sollte außerdem über ein Beitragsbild verfügen. Ein Beitragsbild wird auf der Startseite über dem Titel eures Beitrages angezeigt und macht ihn dadurch visuell ansprechender. Öffnet dafür den Reiter „Beitragsbild“ und klickt auf „Beitragsbild festlegen“. Hier könnt ihr (genau wie wenn ihr ein Bild als Block in euren Beitrag/eure Seite einfügt) entweder ein bereits hochgeladenes Bild auswählen, oder aber ein neues hochladen.

Abschließend solltet ihr noch einstellen, dass eure Sidebar neben eurem Beitrag angezeigt wird, damit sich interessierte Besucher\*innen gleich an euch wenden können und wissen, von wem der Beitrag kommt. ## Beitrag veröffentlichen

Nachdem ihr diese ganzen Einstellungen erledigt habt, könnt ihr euren Beitrag nun veröffentlichen. Klickt dafür auf den blauen Button „Veröffentlichen“ oben rechts. 

import image30 from '@site/static/img/guide-pics/image30.png';

<ImageAside src={image30} third={true}>
Ihr habt die Möglichkeit, einen Beitrag zu einem späteren Zeitpunkt zu veröffentlichen, als ihr ihn schreibt. Klickt dafür in der rechten Seitenleiste unter „Status und Sichtbarkeit“ auf das Datum neben „Veröffentlichen“. Ihr könnt nun ein Datum und eine exakte Uhrzeit einstellen, an dem euer Beitrag automatisch vom System veröffentlich wird.
</ImageAside>

:::info
Bitte nutzt auch hier genauso wie bei den Seiten vor der Veröffentlichung die Vorschaufunktion, um eventuelle Fehler auszumerzen, bevor die Öffentlichkeit Zugriff auf den Beitrag erhält.
:::
